# TableX

TableX is an AI-powered data transformation web application built with Python, Flask, and Google Generative AI (Gemini). It allows users to upload datasets (CSV, JSON), specify data transformations using either natural language prompts or by providing examples, and applies these transformations seamlessly using dynamically generated Python code or batch enrichment API calls.

## Features

- **Upload Datasets**: Upload CSV or JSON files effortlessly.
- **AI-Powered Transformations**: 
  - **Prompt-based**: Describe the transformation you want in natural language (e.g., "Extract the domain from the email address", "Convert to uppercase").
  - **Example-based**: Provide a few source-target examples, and let the underlying LLM figure out the transformation logic.
- **Dynamic Code Generation**: The tool automatically classifies the transformation type (String-based, Numerical, Algorithmic, General) and writes Python functions on-the-fly to execute the transformation without relying on external libraries.
- **Data Enrichment**: Capable of general or real-world knowledge enrichment directly via Gemini.
- **Review & Reattempt**: Review the LLM-generated code or batch results before applying them to your dataset. If a transformation fails, you can quickly reattempt it.
- **Health Reports**: After applying transformations, TableX generates a health report displaying the success rate, value change rate, null values impact, and randomly sampled before-and-after rows.
- **Merge & Export**: Merge multiple transformed datasets into a single file and download the results as CSV, JSON, or a ZIP compilation containing all processed files.

## Tech Stack

- **Backend Architecture**: Python 3, Flask web framework
- **Data Manipulation**: Pandas
- **AI/LLM Engine**: Google Generative AI (`gemini-1.5-flash-latest`)
- **Frontend Layer**: HTML/CSS, Jinja2 Templates

## Getting Started

### Prerequisites

- Python 3.8+
- Your own [Google Gemini API Key](https://aistudio.google.com/app/apikey)

### Installation & Setup

1. **Clone this repository:**
   ```bash
   git clone https://github.com/MaheshChalla2701/TableX.git
   cd TableX
   ```

2. **Install necessary dependencies:**
   ```bash
   pip install pandas flask google-generativeai
   ```

3. **Run the application:**
   ```bash
   python app.py
   ```

4. **Access the application:**
   Open your preferred web browser and navigate to `http://localhost:5000`.

## How It Works

1. **Upload**: Select your file(s) from your local system.
2. **Column Select**: Choose the columns you want to transform.
3. **Define Transformation**: Provide a few examples or write a short prompt describing what needs to be done.
4. **Review**: Check the code generated by the LLM.
5. **Apply**: Execute the transformation over the entire dataset.
6. **Export**: Download your transformed results in your preferred format!
